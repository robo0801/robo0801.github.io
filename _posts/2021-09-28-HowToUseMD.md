Git markdown tip

# 1. Markdown에 글자에 주소(link address) 숨기기

## 인라인 주소 삽입
```
[Google](https://www.google.com/)
```
[Google](https://www.google.com/)

## 참조 링크 삽입
```
주소 삽입 시 문법: \[주소 설명] \[참조번호]
참조 번호 작성 문법: \[참조 번호]: 주소
```
\[Google]\[1]

\[1]: https://www.google.com/
